
# 安装
pip install jupyter

# 更新
pip uninstall jupyter
pip install jupyter

# 启动
创建 D:\jupyter\notebook 目录，或自定义

jupyter notebook --notebook-dir D:\jupyter\notebook

# 快捷键

## Shift+Tab 单击，简略帮助；双击，完整帮助

## Ctrl+S 保存文件

## Shift-Enter : 运行本单元，选中下个单元

## Ctrl-Enter : 运行本单元

## Alt-Enter : 运行本单元，在其下插入新单元
